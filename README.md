# 🎓 Scholargate - Student Agency Platform

> **Empowering students to achieve their academic dreams through scholarships and international opportunities**

---

## ✨ Overview

Scholargate is a comprehensive student agency platform designed to connect students with universities, scholarships, and international opportunities. Our platform streamlines the journey from application to enrollment with professional guidance and comprehensive support.

---

## 🚀 Features

- 📚 **Scholarship Opportunities** - Access to thousands of scholarship programs worldwide
- 🏫 **University Admission** - Streamlined university application process
- 📋 **Visa Processing** - Expert guidance through visa application procedures
- 🏨 **Accommodation Services** - Post-study accommodation and relocation support
- 📞 **Expert Support** - Professional pre-departure consultation and guidance
- ⭐ **Student Reviews** - Transparent feedback from successful applicants
- 📊 **Interactive Dashboard** - Real-time tracking of your application status

## 🛠️ Tech Stack

### Frontend
- **Framework**: React + Vite
- **Language**: JavaScript/TypeScript
- **Build Tool**: Vite
- **Styling**: CSS

### Backend
- **Runtime**: Node.js
- **Initialization**: Express.js (via server.js)

## 📦 Installation & Setup

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Frontend Setup

```bash
cd "Font End"
npm install
npm run dev
```

### Backend Setup

```bash
cd "Back End"
npm install
npm start
```

---

## 🎯 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd "Student Agency"
   ```

2. **Install dependencies** for both frontend and backend

3. **Start the development servers**
   - Backend: Runs on `http://localhost:5000` (or configured port)
   - Frontend: Runs on `http://localhost:5173` (Vite default)

4. **Access the application**
   - Open your browser and navigate to `http://localhost:5173`
